# PresentBox

PresentBox is a modern e-commerce platform built with Next.js 14, designed to aggregate products from multiple sources including Uzum Market, Yandex Market, China, and direct inventory.

## üöÄ Features

- **Multi-Source Aggregation**: Seamlessly display products from:
  - **Uzum Market**: Real-time stock integration and direct links.
  - **Yandex Market**: Integration for broader product selection.
  - **Direct Sales**: Custom inventory management for local products.
  - **China Import**: Specialized section for products sourced from China.
- **Admin Dashboard**: Comprehensive order management system via `/admin`.
- **Responsive Design**: Mobile-first UI built with Tailwind CSS and Lucide React icons.
- **Performance**: Optimized with Next.js App Router and Server Components.
- **Localization**: Support for Uzbek and Russian languages.

## üõ† Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **State Management**: React Hooks & Context

## üì¶ Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/MirobidUsmonov/presentbox.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser.
